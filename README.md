# Xadrez Eletrônico

Projeto desenvolvido para a disciplina **OI25CP-7CPE**.

O projeto consiste em um tabuleiro de xadrez eletrônico baseado em uma matriz 8x8 de reed switches, diodos e peças com ímãs de neodímio, permitindo detectar a presença das peças nas casas do tabuleiro por meio de um microcontrolador ESP32.
